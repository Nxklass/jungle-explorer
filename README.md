# Jungle Explorer

A small 2D platformer built with Python and Pygame.

This project was built to practice core game development concepts such as the game loop, collision handling, menu state management, local save data, and simple level progression.

## Features

- 2 playable levels
- enemy patrol behavior
- moving platform mechanics
- level unlock progression
- settings menu with music and SFX sliders
- local stats and best-time tracking
- optional dev mode toggle
- pause menu and death screen

## Tech Stack

- Python
- Pygame
