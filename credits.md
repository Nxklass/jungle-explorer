Third-party assets are **not automatically covered** by the repository’s code license.  
Before publishing this repository publicly, make sure to do one of the following for every third-party file:

1. document the original source
2. document the license or usage terms
3. add attribution if required
4. remove or replace the asset if the license is unclear
